div2D

PURPOSE

Returns a two-dimensional mimetic divergence operator

SYNOPSIS

function D = div2D(k, m, dx, n, dy)

DESCRIPTION 

 Returns a two-dimensional mimetic divergence operator

 Parameters:
                k : Order of accuracy
                m : Number of cells along x-axis
               dx : Step size along x-axis
                n : Number of cells along y-axis
               dy : Step size along y-axis
