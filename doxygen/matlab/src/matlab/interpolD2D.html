interpolD2D

PURPOSE

Returns a two-dimensional interpolator of 2nd-order

SYNOPSIS

function I = interpolD2D(m, n, c1, c2)

DESCRIPTION 

 Returns a two-dimensional interpolator of 2nd-order
                m : Number of cells along x-axis
                n : Number of cells along y-axis
               c1 : Left interpolation coeff.
               c2 : Bottom interpolation coeff.
