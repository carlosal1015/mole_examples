div3D

PURPOSE

Returns a three-dimensional mimetic divergence operator

SYNOPSIS

function D = div3D(k, m, dx, n, dy, o, dz)

DESCRIPTION 

 Returns a three-dimensional mimetic divergence operator

 Parameters:

                m : Number of cells along x-axis
               dx : Step size along x-axis
                n : Number of cells along y-axis
