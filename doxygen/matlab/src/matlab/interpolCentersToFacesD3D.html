PURPOSE

3D interpolation from centers to faces

SYNOPSIS

function I = interpolCentersToFacesD3D(k, m, n, o)

DESCRIPTION 

 3D interpolation from centers to faces
 logical centers are [1 1.5 ... m-0.5 m]x[1 1.5 ... n-0.5 n]x[1 1.5 ... o-0.5 o]
