PURPOSE

SYNOPSIS

function [J, Xe, Xn, Xc, Ye, Yn, Yc, Ze, Zn, Zc] = jacobian3D(k, X, Y, Z)

DESCRIPTION 

 Returns:
                J : Determinant of the Jacobian
               Xe : dx/de metric
               Xn : dx/dn metric
               Xc : dx/dc metric
               Ye : dy/de metric
               Yn : dy/dn metric
               Yc : dy/dc metric
               Ze : dz/de metric
               Zn : dz/dn metric
               Zc : dz/dc metric

 Parameters:
                k : Order of accuracy
                X : x-coordinates (physical) of meshgrid
                Y : y-coordinates (physical) of meshgrid
                Z : z-coordinates (physical) of meshgrid
