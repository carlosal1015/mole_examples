jacobian2D

PURPOSE

Returns:

SYNOPSIS

function [J, Xe, Xn, Ye, Yn] = jacobian2D(k, X, Y)

DESCRIPTION 

 Returns:
                J : Determinant of the Jacobian (XeYn - XnYe)
               Xe : dx/de metric
               Xn : dx/dn metric
               Ye : dy/de metric
               Yn : dy/dn metric

 Parameters:
                k : Order of accuracy
                X : x-coordinates (physical) of meshgrid
                Y : y-coordinates (physical) of meshgrid
