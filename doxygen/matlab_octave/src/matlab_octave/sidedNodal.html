SYNOPSIS

function S = sidedNodal(m, dx, type)

DESCRIPTION 

 Returns a m+1 by m+1 one-dimensional sided approximation for uniformly
 spaced data points. This function is handy for advective terms.

 Parameters:
                m : Number of cells
               dx : Step size
             type : 'backward', 'forward' or 'centered'
