gradPeriodic

PURPOSE

Returns a m by m+2 one-dimensional mimetic gradient operator

SYNOPSIS

function G = gradPeriodic(k, m, dx)

DESCRIPTION 

 Returns a m by m+2 one-dimensional mimetic gradient operator
 when the boundary condition is periodic

 Parameters:
                k : Order of accuracy
                m : Number of cells
               dx : Step size
