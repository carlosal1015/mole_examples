divPeriodic

PURPOSE

Returns a m+2 by m one-dimensional mimetic divergence operator

SYNOPSIS

function D = divPeriodic(k, m, dx)

DESCRIPTION 

 Returns a m+2 by m one-dimensional mimetic divergence operator
 when the boundary condition is periodic

 Parameters:
                k : Order of accuracy
                m : Number of cells
               dx : Step size
