0017 function L = lap2D(k, m, dx, n, dy)
0018 % Returns a two-dimensional mimetic laplacian operator
0019 %
0020 % Parameters:
0021 %                k : Order of accuracy
0022 %                m : Number of cells along x-axis
0023 %               dx : Step size along x-axis
0024 %                n : Number of cells along y-axis
0025 %               dy : Step size along y-axis

0029
